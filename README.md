# ResizeKB

ResizeKB is a free online image resize and compression tool. It allows users to resize images to exact file sizes such as 20KB, 50KB, 100KB, and 200KB.

## Features

- Resize image to 20KB
- Resize image to 50KB
- Resize image to 100KB
- Resize image to 200KB
- Browser-based image processing
- No server upload required
- Fast and mobile-friendly

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Goal

The goal of this project is to create a simple, fast, and SEO-friendly online image resize tool for users who need exact KB image sizes for forms, exams, documents, and online applications.
